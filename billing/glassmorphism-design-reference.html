🚀 Key Features

  • Semi-transparent dark background with smooth vertical gradient
  • Frosted glass effect using backdrop-filter blur
  • Floating shadow effect for depth perception
  • Rounded right corners (20px border-radius)
  • White text with varying opacity levels
  • Subtle highlight on hover with transform animation
  • Pink gradient background for active menu items
  • Smooth 0.3s ease transitions on all interactions
  • Custom scrollbar with pink gradient theme
  • Responsive design for mobile, tablet, and desktop
  • Collapsible dropdown menus with animations
  • Icon animations on hover (scale 1.1)
  • Dot indicators for submenu items
  • Chevron rotation for dropdown states
  • Production-ready pure HTML + CSS
  • No external framework dependencies

💻 Core CSS Code

.sidebar {
    background: linear-gradient(
        180deg,
        rgba(30, 30, 40, 0.85) 0%,
        rgba(40, 40, 55, 0.85) 50%,
        rgba(50, 50, 65, 0.85) 100%
    );
    backdrop-filter: blur(20px);
    -webkit-backdrop-filter: blur(20px);
    border-right: 1px solid rgba(255, 255, 255, 0.1);
    box-shadow: 
        4px 0 24px rgba(0, 0, 0, 0.3),
        inset -1px 0 0 rgba(255, 255, 255, 0.05);
    border-radius: 0 20px 20px 0;
    transition: all 0.3s ease;
}
.sidebar-item.active {
    background: linear-gradient(
        135deg,
        rgba(255, 105, 180, 0.3) 0%,
        rgba(219, 39, 119, 0.3) 100%
    );
    border-left-color: #ff69b4;
    box-shadow: 
        0 4px 15px rgba(255, 105, 180, 0.4),
        inset 0 1px 0 rgba(255, 255, 255, 0.1);
}
